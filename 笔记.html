redux 的一些概念

api: 一共有三个api import {createStore} from 'redux'
  • 1. createStore(reducer) 创建一个状态容器
    const store = createStore(reducer);
  • 2. subscribe 这个api是在store上的，去订阅数据的更新。 store.subscribe(function () { })
  • 3. 获取数据 store.getState() 返回store当中最新的数据。
  • 4. dispatch dispatch是在store对象上的。通过store.dispatch(action)去更改store当中的数据。

概念:

  • 1. reducer reducer是一个纯的函数，不能在里边操作外部的数据，而是应该根据传入的action的不同，去返回新的数据。reducer不一定要求用switch语法。
  • 2. action action是指一个动作，唯一改变store当中数据的方法，是去dispatch出发一个action。那么action在代码当中是一个纯对象。 const action = {type: "INCREMENT"}
  • 3. actionCreater 是指action的创造函数，function （item） { return {type:'', item} }
